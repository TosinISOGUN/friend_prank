# Friendship Request App

An interactive, playful web application designed to ask a friend (Naomi) a very important question.

## Features

- **Interactive Buttons**: The "No" button jumps away from the cursor, while the "Yes" button grows with each attempt.
- **Celebration State**: Full-screen confetti and a heartwarming message upon acceptance.
- **Modern UI**: Built with a sleek, responsive design using React and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js & npm installed.

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd friendship_request
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- **Vite**: Frontend tooling
- **React**: UI library
- **TypeScript**: Static typing
- **shadcn/ui**: Component library
- **Tailwind CSS**: Utility-first CSS framework
- **Canvas-Confetti**: Particle animations
